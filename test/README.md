# Test Files

Place test images here for testing the virtual camera.

## Recommended Test Images

1. **test_640x480.png** - Standard resolution test
2. **test_1920x1080.png** - HD resolution test
3. **test_pattern.jpg** - Test pattern for calibration

## Creating Test Images

You can create simple test images using online tools or any image editor:

- https://placeholder.com/
- GIMP (free)
- Paint.NET (free)
- Photoshop

## Test Checklist

- [ ] 640x480 PNG loads correctly
- [ ] 1920x1080 PNG loads correctly
- [ ] JPG files load correctly
- [ ] Camera appears in video applications
- [ ] Image displays correctly (not upside down or distorted)
- [ ] Frame rate is smooth (30 fps)
- [ ] Multiple applications can access the camera simultaneously

## Known Test Applications

- Windows Camera app
- Zoom
- Microsoft Teams
- OBS Studio
- VLC Media Player (Open Capture Device)
